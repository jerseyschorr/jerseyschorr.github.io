# 📘 Kitzur Shulchan Aruch Yomi Tracker

A clean, mobile‑friendly, PWA‑enabled web app for tracking the **Kitzur Shulchan Aruch Yomi** cycle.

Hosted at: **https://jerseyschorr.github.io**

---

## ✨ Features

- Daily learning tracker (Hebrew + English)
- Full 365‑day cycle (ArtScroll‑style ranges)
- Streak counter and stats card
- Dark mode
- Loading animation
- PWA (offline, installable)
- Backup/export/import of progress
- Shabbos Mode (disables interaction)
